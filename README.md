# Проект: Landing Page (alekhtml)

Легкий и адаптивный лендинг пейдж с настроенной формой обратной связи.

## 📂 Структура проекта

```text
alekhtml/
├── assets/                 # Ресурсы фронтенда
│   ├── css/
│   │   └── custom.css      # Стили (Flexbox/Grid, адаптив)
│   ├── js/
│   │   └── main.js         # Логика: валидация формы, AJAX-запрос
│   └── images/             # Оптимизированная графика
├── api/
│   └── sendmail.php        # Backend: обработка POST-запросов и отправка email
├── index.html              # Главная страница (Semantic HTML5)
└── privacy.html            # Страница политики конфиденциальности
